Découverte du JavaScript

Introduction

1 : Que signifie le nom JavaScript ?
Réponse : A l'époque, pendant les années 1990, Java était un langage extrêmement populaire. Le créateur de JavaScript a repris "Java" pour attirer la communeauté utilisant Java. Le nom : "Script" indique que c’est un langage de scriptage, c’est-à-dire un langage conçu pour être exécuté directement, souvent pour automatiser ou ajouter des comportements à une page web.

2 : À quoi sert JavaScript dans un site web ?
Réponse : Dans un site web, le JavaScript sert à rendre les pages interactives et dynamiques.

3 : Dans quel endroit peut-on écrire du JavaScript dans une page HTML ?
Réponse : On peut écrire du JavaScript dans une page HTML avec la balise scripten la plaçant soit dans la section head soit à la fin du body.

4 : Quelle est la balise HTML utilisée pour écrire du JavaScript ?
Réponse :La balise HTML utilisée pour écrire du JavaScript est script

5 : Peut-on aussi écrire le JavaScript dans un fichier séparé ? Si oui, quelle est l’extension de ce fichier ?
Réponse : Oui, on peut aussi écrire le JavaScript dans un fichier séparé : on utilise alors l'extension .js

6 : Que fait la commande suivante : console.log("Bonjour") ?
Réponse : La commande console.log("Bonjour") fait afficher le texte "Bonjour" dans la console.

7 : Que fait la fonction suivante : alert("Bonjour") ?
Réponse : La fonction alert("Bonjour") fait afficher une boîte de dialogue dans le navigateur contenant "Bonjour"

8 : Qu’est-ce qu’une variable en JavaScript ? Une variable est un objet repéré par son nom, pouvant contenir des données, qui pourront être modifiées lors de l'exécution du programme.

9 : À quoi servent les mots-clés suivants : let ; const ?
Réponse : Les mots-clés let et const servent à déclarer des variables. Ils permettent de créer un espace mémoire pour stocker une valeur.

10 : Que fait ce code : let nom = "Nour"; alert(nom);?
Réponse : Ce code va créer une fenêtre pop-up dans le navigateur où apparaitra "Nour"

11 : Qu’est-ce qu’une fonction en JavaScript ?
Réponse : Une fonction est une procédure JavaScript, un ensemble d'instructions effectuant une tâche ou calculant une valeur.

15 : Pourquoi JavaScript est-il important dans les sites web modernes ?
Réponse : Car il permet d'ajouter des fonctionnalités interactives aux pages web, telles que les menus déroulants, les animations ou encore les formulaires dynamiques.
