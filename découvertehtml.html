Découverte du langage HTML

Questionnaire sur le HTML

1 : Quand tu ouvres un site web dans ton navigateur (par exemple Wikipédia ou YouTube), que se passe-t-il selon toi entre le moment où tu tapes l’adresse et le moment où la page s’affiche ? Réponse : Selon moi, le navigateur va charger l’adresse du site (https://www.wikipedia.org) pour l’afficher dans notre fenêtre de navigateur.

2 : Qu’est-ce qu’une page web ? Réponse : Une page Web est une unité de consultation du World Wide Web, qui est conçue pour être consultée sur un navigateur web , et est constituée généralement d’un document rédigé en HTML.

3 : Qu’est-ce qu’un site web ? Quelle est la différence entre un site web et une page web ? Réponse : Un site web est un ensemble de pages web et de ressources, qui sont reliées entre elles avec des hyperliens et qui sont accessibles via un navigateur internet. La différence avec une page web est que le site web est accessible via le réseau mondial internet.

4 : Qu’est-ce qu’un navigateur web ? Donne quelques exemples de navigateurs. Réponse : Un navigateur internet est un logiciel conçu pour consulter le web. Il en existe plusieurs, comme par exemple Chrome qui appartient à Google, Edge qui appartient à Microsoft, Firefox qui appartient à Mozilla Foundation, ou encore Safari qui appartient à Apple.

5 : Qu’est-ce que le HTML ? À quoi sert-il ? Réponse : Le HTML est un langage de balisage qui sert à écrire des pages web. Il est très utilisé en informatique, et c’est un langage ouvert.

6 : Que signifie l’acronyme HTML ? Réponse : L’acronyme HTML signifie Hyper Text Markup Language.

7 : Pourquoi dit-on que le HTML est un “langage de balises” ? Réponse : On dit que le HTML est un langage de balise car il représente une catégorie de langages spécialisés dans l’enrichissement textuel.

8 : Qu’est-ce qu’une balise en HTML ? Réponse : Il utilise des balises qui sont des unités syntaxiques qui délimitent une séquence de caractères ou qui marque une position précise à l’intérieur d’un flux de caractères.

9 : À quoi sert une balise ouvrante et une balise fermante ? Réponse : Une balise « ouvrante » marque le début d’un élément. Par exemple le début d’un paragraphe. Une balise « fermante » marque la fin de cet élément.

10 : À ton avis, à quoi servent ces balises ? h1 p a img Réponse : La balise h1 sert à définir le texte principal sur une page. La balise h1 sert à écrire un paragraphe de texte. La balise a sert à créer un lien hypertexte vers une autre page, un site, ou un fichier. Enfin, la balise img sert à afficher une image sur la page.

11 : Quand tu lis une page web, vois-tu le code HTML ou seulement le résultat ? Pourquoi ? Réponse : Quand je lis une page Web, je vois seulement le résultat, car le HTML sert à décrire la structure de la page, et le navigateur l’interprète, en traduisant le code en page visuelle avec du texte des boutons et des images.

12 : Comment peut-on afficher le code source d’une page web dans un navigateur ? Réponse : : Il suffit d’appuyer sur CTRL + U

13 : Quelle est la différence entre : HTML CSS JavaScript Réponse : Le HTML sert à structurer le contenu d’une page web. Le CSS sert à styliser/gérer l’apparence du site. Le JavaScript est un langage de programmation qui sert à rendre le site interactif.

14 : À ton avis, quel est le rôle de chacun dans la construction d’un site web ? Réponse : Chacun a son propre rôle : l’un stylise le site, l’autre le rend interactif, et le dernier sert à le structurer.

15 : Qu’est-ce qu’un éditeur de code ? Pourquoi en a-t-on besoin pour programmer ? Réponse : : Un éditeur de code est un logiciel qui permet d’écrire et de modifier du code source. Si on avait pas d’éditeur de code, ce serait beaucoup plus difficile.
