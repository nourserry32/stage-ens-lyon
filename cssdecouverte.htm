Questionnaire CSS

1 : Que signifie l’acronyme CSS ? Réponse : L'acronyme CSS signifie Cascading Style Sheets

2 : À quoi sert le CSS dans la création d’un site web ? Réponse : Le CSS permet de mettre en forme les éléments HTML

3 : Quelle est la différence entre HTML et CSS ? Réponse : Le HTML sert à structurer une page web, tandis que le CSS sert à styliser cette page web.

4 :Pourquoi sépare-t-on généralement le contenu (HTML) et la mise en forme (CSS) ? Réponse : On sépare le contenu HTML et le contenu CSS car cela permet de maintenir un code plus propre et mieux organisé qui facilite les modifications des développeurs.

5 : Qu’est-ce qu’une règle CSS ? Réponse : Les règles sont des instrcutions qui indiquent au CSS comment se comporter et comment afficher un ou plusieurs éléments HTML sur une page Web.

6 : Une règle CSS est généralement composée de deux parties. Lesquelles ? Réponse : Les deux parties composant une règle CSS sont un sélecteur et un bloc de déclaration.

7 : Qu’est-ce qu’un sélecteur CSS ? Réponse : Un sélecteur CSS est une règle qui permet de choisir quels éléments HTML seront stylisés par une déclaration CSS

8 : Donne quelques exemples de sélecteurs CSS (par exemple pour sélectionner une balise, une classe ou un id). Réponse : Pour sélectionner une balise : p { color: blue; } ; Pour sélectionner une classe : .bouton { background: red; } ; Pour sélectionner un ID : #logo { width: 100px; } ;

9 : Quelle est la différence entre : un id une classe (class) ? Réponse : L'ID sert à sélectionner UN SEUL élément précis d'une page web. La classe sert à appliquer le même style à plusieurs éléments.

10 : Que fait la propriété CSS color ? Réponse : La propriété CSS color permet de définir la valeur de couleur du texte d'un élément et de ses décorations de texte.

11 : Que fait la propriété CSS background-color ? Réponse : La propriété background-color sert à définir la couleur d'arrière-plan d'un élément.

12 : Que fait la propriété CSS font-size ? Réponse : La propriété font-size permet de déterminer la taille du texte.

13 : Que fait la propriété CSS margin ? Réponse : La propriété margin crée de l'espace autour de l'élément, à l'extérieur de ses bordures.

14 : Que fait la propriété CSS padding ? Réponse : La propriété padding crée de l'espace à l'intérieur de l'élément, entre son contenu et sa bordure.

15 : Que signifie l’expression “feuille de style” ? Réponse : L'expression feuille de style désigne un document ou un ensemble de règles qui définit l’apparence visuelle d’une page web.

16 : De quelles façons peut-on ajouter du CSS à une page HTML ? Réponse : On peut ajouter du CSS à une page HTML en style de ligne, c'est-à-dire que l'on l'ajoute directement dans le code source. On peut également l'ajouter en feuille de style interne au début du document HTML. On peut aussi l'ajouter en feuille de style externe, c'est-à-dire qu'on l'ajoute dans un fichier CSS externe.

17 : Quelle est la différence entre : CSS interne CSS externe Réponse : La différence entre du CSS interne, externe, et inline vient de leur emplacement et de leur champ d'application.

18 : Pourquoi est-il généralement préférable d’utiliser un fichier CSS externe ? Réponse : Il est généralement préférable d'utiliser un fichier CSS externe pour une meilleure lisibilité, une meilleure maintenance et un meilleur partage des styles parmi plusieurs fichiers HTML.

19 : Si une section HTML possède id="introduction", comment peut-on la sélectionner en CSS ? Réponse : Pour sélectionner une section HTML qui possède id=introduction, on utilise le dièse (#).

20 : Trouve un exemple simple de code CSS et explique ce que fait chaque ligne Exemple simple de code CSS (généré avec ChatGPT) : body { background-color: lightblue; font-family: Arial, sans-serif; } h1 { color: darkblue; text-align: center; } p { font-size: 18px; color: black }

Body est un sélecteur CSS. Il cible la balise body du HTML. background-color définit la couleur de fond. lightblue signifie bleu clair : toute la page aura un fond bleu clair. font-family définit la police d’écriture. Arial est la police principale. sans-serif est une police de secours si Arial n’est pas disponible.
